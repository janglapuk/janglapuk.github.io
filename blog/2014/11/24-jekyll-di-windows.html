Jekyll di Windows

Dalam keseharian, aku menggunakan Mac untuk kegiatan yang berhubungan dengan internet, termasuk untuk meng-update blog ini menggunakan jekyll. Hanya saja, sesekali PC Windows-lah yang aku gunakan. Meng-install jekyll di Mac sudah tidak terlalu masalah bagiku, karena ruby yang digunakan jekyll sebagai interpreter di Mac, sudah ter-include secara default. Beda halnya jika di lingkungan Windows.

Tidak usah banyak kata-kata lagi, berikut ini langkah menggunakan jekyll di Windows:

1. Ruby

Karena jekyll menggunakan ruby sebagai bahasa pemrogramnya, langkah utama yang perlu dilakukan adalah memasang ruby pada PC Windows.

Secara resmi, website ruby adalah https://www.ruby-lang.org. Hanya saja jika anda berniat menggunakan ruby di lingkungan Windows, maka anda harus ke website http://rubyinstaller.org/downloads/ untuk mengunduhnya, kemudian unduh rubyinstaller-2.1.5.exe untuk mesin berarsitektur 32 bit, atau rubyinstaller-2.1.5-x64.exe jika berarsitektur 64 bit.

Pastikan bahwa opsi seperti tanda gambar di atas benar-benar dicentang (checked).

2. Ruby Development Kit (DevKit)

Langkah berikutnya adalah mengunduh Ruby Development Kit di halaman website yang sama, yaitu http://rubyinstaller.org/downloads/. Sama seperti disampaikan sebelumnya, sesuaikan dengan arsitektur prosesor pada PC anda.

Jalankan installer tersebut dan arahkan ke lokasi tujuan extract yang mudah untuk ditemukan, seperti C:\RubyDevKit.

Kemudian, lakukan proses berikut menggunakan command prompt untuk jalankan perintah berikut:

cd C:\RubyDevKit
ruby dk.rb init
ruby dk.rb install

Dari proses perintah di atas, setidaknya hasilnya kurang lebih seperti gambar di bawah ini:

3. Jekyll

Langkah berikut ini adalah memasang program jekyll pada PC anda, hanya saja anda tidak perlu mengunduhnya pada website tertentu, cukup menggunakan salah satu aplikasi bawaan ruby yaitu RubyGem (perintah: gem).

Dengan menggunakan command prompt, silakan jalankan perintah berikut:

gem install jekyll

4. Rouge

Kemudian, dalam menggunakan jekyll anda wajib menggunakan Markdown untuk mempercantik tulisan anda. Tentunya akan banyak kode-kode program yang akan anda selipkan nantinya. Karena itu, rouge memiliki peran sebagai beautifier kode-kode anda (syntax highlighter).

Sebenarnya, secara default jekyll menggunakan Pygments sebagai syntax highlighter. Hanya saja untuk menggunakan Pygments, anda perlu memasang Python agar dapat menggunakannya. Aku pikir itu saat ini Rouge yang paling cocok digunakan karena tidak memerlukan berbagai macam dependensi seperti halnya Pygments.

Untuk memasang Rouge, lakukan langkah seperti memasang jekyll, yaitu dengan menggunakan gem pada command prompt:

gem install rouge

Catatan: Nantinya setelah sebuah project dibuat, anda cukup mengubah/menambahkan satu entri pada file <project-name>\_config.yml, yaitu pada entri:

highlighter: rouge

5. WDM

Secara default, jekyll memiliki fitur auto-regeneration yang secara otomatis memantau (watch) perubahan source pada jekyll dan kemudian re-build situs (site) anda.

Namun pada Windows, anda perlu memasang satu alat tambahan yaitu wdm. Dengan menggunakan perintah berikut:

gem install wdm

6. Run!

Akhirnya, kita sudah berada di langkah terakhir, yaitu mencoba membuat dan menjalankan website menggunakan jekyll.

Anggap saja anda memiliki sebuah direktori khusus untuk membuat project anda dengan jekyll. Di sini saya menyatakan direktori project adalah D:\project\website. Silakan anda sesuaikan sesuai kebutuhan anda.

Pada command prompt anda, lakukan perintah berikut:

D:
cd D:\project\website

Kemudian, buatlah satu project dengan nama myblog dengan menggunakan jekyll pada command prompt:

jekyll new myblog

Maka akan ada sebuah direktori dengan nama myblog di dalam root direktori project anda.

Kemudian, masuklah ke dalam direktori myblog, dan jalankan project tersebut menggunakan jekyll dengan perintah:

cd myblog
jekyll serve

7. Post

Berikut ini mungkin hanya sebagai tambahan saja, bagaimana membuat sebuah post pada jekyll. Yang anda perlukan hanyalah pengetahuan tentang sintaks pada Markdown dan format penamaan sebuah file post. (Untuk mengetahui bagaimana sintaks pada Markdown, silakan periksa pada bagian Referensi di bawah.)

Sebagai contoh: Gunakan text editor kesayangan anda, kemudian buatlah tampak seperti di bawah ini:

---
layout: post
title:  "Post Pertama"
date:   2014-11-24 13:50:00 +0800
categories: Blog
tags: jekyll hello-world
---
Ini adalah halaman *post* saya yang pertama menggunakan **jekyll**.

Simpanlah file tersebut di direktori myblog\_posts dengan format file tahun-bulan-tanggal-judul.markdown seperti:

2014-11-24-post-pertama.markdown.

Kemudian, silakan refresh halaman blog anda, secara otomatis jekyll akan meng-update setiap perubahan pada file-file project anda.
